  MidJourney Prompt Automation Tool v1.0

════════════════════════════════════════

🚀 QUICK START:
--------------
1. Double-click "mid_journey_automation.exe"
2. If Windows shows a security warning:
   - Click "More info"
   - Click "Run anyway"
3. Upload your DOCX file with prompts
4. Configure your settings
5. Click START and switch to MidJourney window
6. Let it automate everything!

⚙️ SYSTEM REQUIREMENTS:
-----------------------
- Windows 7/8/10/11
- No installation required!
- No Python needed!

📖 HOW TO USE:
--------------
1. UPLOAD: Click "Select DOCX File" and choose your prompts file
2. CONFIGURE: Set typing speed, delays, and batch settings
3. START: Click the green START button
4. SWITCH: Quickly switch to your MidJourney window
5. CLICK: Click in the text input box
6. RELAX: The tool does everything automatically!

🎯 FEATURES:
------------
✅ Batch Mode - Send prompts in groups with breaks

✅ Adjustable Speed - Instant to normal typing

✅ Pause/Resume - Full control during automation

✅ Progress Tracking - Know exactly where you are

✅ CSV Export - Automatic conversion from DOCX

⚠️ IMPORTANT NOTES:
-------------------
- Keep MidJourney window active while running
- Don't touch keyboard/mouse during automation
- Emergency stop: Move mouse to top-left corner
